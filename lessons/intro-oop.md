## Creating Object (Instantiation)

‡πÄ‡∏£‡∏≤‡∏•‡∏≠‡∏á‡∏°‡∏≤‡∏î‡∏π‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏Å‡∏±‡∏ô

```python
class User:
  def __init__(self, name, score):
    self.name = name
    self.score = score

  def increase_score(self):
    self.score += 1

  def get_score(self):
    return self.score
```
